# YouTube Clone

## Overview

This is a YouTube clone application built with React, Tailwind CSS, Redux, and Vite. The app aims to replicate core features of YouTube, including video search, playback, and comment threads.

## Features

- **Video Search**: Search for videos using YouTube's API.
- **Video Playback**: Watch video content in a responsive player.
- **Comments Section**: View and manage comments on videos.
- **Responsive Design**: Styled with Tailwind CSS for a responsive user experience.

## Technologies Used

- **React**: A JavaScript library for building user interfaces.
- **Tailwind CSS**: A utility-first CSS framework for styling.
- **Redux**: A state management library for managing application state.
- **YouTube Data API**: For fetching video and comment data.
